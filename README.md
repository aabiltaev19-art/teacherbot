# teacherBot

TeacherBot — это чат-бот-преподаватель, который задает вопросы и проверяет ответы.

## Цель
Создать бота, который обучает и тестирует знания студентов.

## Возможности
- задает вопросы
- проверяет ответы
- выдает результат

## Технологии
- HTML
- CSS
- JavaScript
- GitHub

## Результат
Создан бот-преподаватель с логикой диалога.
